## Where does this problem come from?

> In 2023, AutoGPT and the ReAct framework brought the Agent concept into the mainstream: large models were no longer just answering questions, but could call tools, plan tasks, and interact continuously with the environment. Agents became the bridge connecting model capabilities to real-world actions.

## Learning Objectives

After completing this module, you should be able to answer:
1. How does the ReAct framework enable models to both reason and act?
2. What engineering patterns exist for Agent memory, tool use, and multi-agent collaboration?
3. How do you evaluate the reliability and safety of an Agent system?

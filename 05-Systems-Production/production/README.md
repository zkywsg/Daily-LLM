# 生产系统 (Production Systems)

[English](README_EN.md) | [中文](README.md)

## 这个问题从哪来

> 2023–2024年，大模型从实验室demo走向真实业务场景：企业搜索、代码助手、客服机器人。但"模型能跑"不等于"系统能交付"——延迟、成本、安全、可维护性成为决定项目成败的关键。

## 学习目标

完成后你应能回答：
1. 企业搜索与通用搜索在架构上有哪些本质差异？
2. 代码助手和对话系统各自的核心优化目标是什么？
3. 生产级 LLM 应用需要哪些质量与安全保障？

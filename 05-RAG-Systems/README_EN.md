# Phase 5: RAG & Agent Systems

**[English](README_EN.md) | [中文](README.md)**

## Overview

This phase focuses on building advanced retrieval systems and autonomous agents. We move beyond simple prompts to creating systems that can reason, retrieve external knowledge, and interact with the world.

## Structure

### 1. [RAG Foundations](rag-foundations/README.md)
Core components of Retrieval-Augmented Generation systems.
- **Chunking Strategies**: How to split text effectively.
- **Embedding Models**: Converting text to vectors.
- **Reranking**: Improving retrieval precision.
- **Retrieval Evaluation**: Metrics like MRR and NDCG.

### 2. [Vector Databases](vector-databases/README.md)
The storage engine for semantic search.
- **Concepts**: HNSW, IVF, Scalar Quantization.
- **Tools**: Milvus, Pinecone, Weaviate, Chroma.

### 3. [Agents](agents/README.md)
Building autonomous systems.
- **Architecture**: Planning, Memory, Tools.
- **Patterns**: ReAct, Plan-and-Solve.
- **Multi-Agent Systems**: Collaboration patterns.
- **Tool Use**: Function calling and API integration.

### 4. [Production Systems](production/README.md)
Real-world application patterns.
- **Enterprise Search**: Secure, scalable search.
- **Code Assistants**: Copilot-like systems.
- **Conversational AI**: Context-aware chatbots.
- **Quality & Security**: Guardrails and governance.

# 企业搜索 (Enterprise Search)

## 概述
企业搜索帮助员工在内部文档、邮件和数据库中查找信息。

## 核心功能
- 统一搜索所有公司数据 (Unified search across all company data)
- 访问控制和权限管理 (Access control and permissions)
- 查询的语义理解 (Semantic understanding of queries)
- 基于用户角色的个性化 (Personalization based on user role)

## 实现方式
结合 BM25 进行关键词匹配和向量搜索实现语义理解。

## 之前卡在哪

NLP 任务接口分裂，研究者还不确定单纯扩大语言模型是否会持续带来能力提升。

## 发生了什么

GPT-2 展示大规模自回归生成能力，T5 把多种 NLP 任务统一为文本到文本格式。

## 解决了什么

规模化训练和统一任务接口成为新的主线。

## 留下了什么新问题

参数、数据和算力快速膨胀，评估与安全问题开始变得突出。

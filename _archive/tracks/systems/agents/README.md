# 智能体 (Agents)

[English](README_EN.md) | [中文](README.md)

## 这个问题从哪来

> 2023年，AutoGPT 和 ReAct 框架让 Agent 概念破圈：大模型不再只是回答问题，而是能够调用工具、规划任务、与环境持续交互。Agent 成为连接模型能力与真实世界行动的桥梁。

## 学习目标

完成后你应能回答：
1. ReAct 框架如何让模型同时具备推理与行动能力？
2. Agent 的记忆、工具调用与多 Agent 协作各有哪些工程模式？
3. 如何评估一个 Agent 系统的可靠性与安全性？
